校园图书借阅自助管理系统运行说明

环境要求：JDK、Maven、Node.js、MySQL 8.0 已安装并配置环境变量。

1. 初始化数据库
   mysql -h127.0.0.1 -P3306 -uroot -p123456 < database.sql

2. 启动后端
   cd springboot_back && mvn spring-boot:run

3. 启动前端
   cd vue_front && npm install && npm run dev

4. 访问地址
   前端：http://localhost:5173
   后端：http://localhost:8080

5. 默认账号
   管理员：admin / 123456
   普通用户：student / 123456
   逾期测试用户：badguy / 123456
